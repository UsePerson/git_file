Command

  • 1s
    • 查看檔案及子目錄

  • 1s -la
    • -l 列出詳細資料
    • -a 列出隱藏資料

  • pwd
    • print work directory，印出目前工作目錄

  • cd
    • change directory，移動進入資料夾

  • mkdir
    • make directory，創建新資料夾

  • cp
    • copy，複製檔案

  • mv
    • move (rename) files，移動檔案或是重新命名檔案

  • rm
    • remove file，刪除檔案

  • touch
    • 用來更新已存在文件的 timestamp 或是 新增空白檔案

  • cat
    • 將文件印出在 terminal 上

  • tail
    • 顯示檔案最後幾行內容

  • tail -f file
    • 持續顯示更新內容，常用於 web server 看 log debug 使用

  • more
    • 將檔案一頁頁印在終端機上，可以使用上下移動換頁，按 q 離開

  • file
    • 檢查檔案類型
